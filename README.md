# yukonChromeExtension

这是一个功能强大的个人专属浏览器助手，集成了视频倍速控制、自动切集、代理管理以及网页限制解除等多种实用功能。

## 安装说明

1. 下载并解压本项目代码。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式” (Developer mode)**。
4. 点击 **“加载已解压的扩展程序” (Load unpacked)**，选择本项目所在的文件夹。

## 核心功能

- **视频倍速增强**:
  - **长按快进**: 长按 `ArrowRight` (右方向键) 视频以 2.0x 倍速播放，松开即恢复。
  - **自动起播**: 长按倍速时若视频处于暂停状态，插件将自动尝试恢复播放。
  - **智能搜索**: 支持深度搜索 Shadow DOM 及 iframe 内部的播放器。
- **快捷切集**:
  - 使用快捷键 `[` (上集) / `]` (下集) 自动查找并点击页面上的导航按钮。
  - 支持启发式文本匹配与 URL 模式预测。
- **超级复制粘贴**:
  - 解锁网页右键限制、选择限制以及复制粘贴限制。
  - 智能拦截干扰事件，同时确保不影响播放器等正常交互。
- **代理控制器**:
  - **一键切换**: 在“系统代理”与“直接连接 (No Proxy)”模式间快速切换。
  - **解决冲突**: 有效解决部分网站因代理导致的“不提供服务”或连接缓慢问题。
- **UI/UX 优化**:
  - **精美指示器**: 倍速或快进时在屏幕上方显示精致的磨砂玻璃质感提示。
  - **状态角标**: 插件弹出页（Popup）头部带有状态角标，支持直接点击切换状态，操作更高效。

## 设置管理

1. 在扩展程序列表或工具栏点击图标进入设置页。
2. **白名单管理**: 自定义哪些域名可以使用视频控制功能。
3. **状态切换**: 通过开关或标题栏的快捷角标，控制代理模式与超级复制功能的开启/关闭。

## 实现原理

- **HTML5 Video API**: 标准化控制媒体播放速率。
- **MutationObserver**: 持续监听并清理网页干扰元素（如特定横幅广告）。
- **Chrome Proxy API**: 动态修改浏览器代理配置。
- **Event Interception**: 捕获并停止阻止用户操作的 DOM 事件。
