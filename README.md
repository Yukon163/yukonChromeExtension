# Cycani 视频倍速播放插件

这是一个简单的 Chrome 浏览器插件，专门为 `cycani.org` 设计。

## 安装说明
1. 下载并解压本项目代码。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式” (Developer mode)**。
4. 点击 **“加载已解压的扩展程序” (Load unpacked)**，选择本项目所在的文件夹。

## 功能
- **长按右方向键 (ArrowRight)**: 视频将以 2.0 倍速播放，松开恢复。
- **快捷键 `[` / `]`**: 自动查找并点击页面上的“上一集”或“下一集”按钮。
- **自定义白名单**: 在插件设置页可以以列表形式管理生效的域名。
- **精美 UI 提示**: 采用磨砂玻璃质感的右上角提示框，字体适中，不干扰观影。
- **针对 cycani.org 的直连优化**: 插件内置了代理规则，访问 `cycani.org` 及其子域名时将自动跳过代理，采用直连（No Proxy）模式，解决因代理导致的“不提供服务”问题。

## 设置管理
1. 在 `chrome://extensions/` 页面点击本插件的 **“详情” (Details)** -> **“扩展程序选项” (Extension options)**。
2. 你可以添加新域名、删除已有域名。
3. **直连优化开关**：你可以手动开启或关闭针对 `cycani.org` 的直连功能。如果开启后发现其他网站代理失效，请关闭此项并在你的专业代理插件（如 SwitchyOmega）中手动配置。

## 实现原理
本插件通过以下方式实现，具有极高的通用性：
1. **倍速播放**: 利用 HTML5 Video 标准 API，支持 Shadow DOM 深度搜索。
2. **上下集切换**: 启发式文本匹配逻辑。
3. **跨域注入**: 支持 iframe 内部播放器。
4. **智能代理控制**: 只有在设置页开启开关后，才会利用 `chrome.proxy` API 实现特定域名的直连加速。关闭开关会立即释放代理控制权，交还给系统或其他插件。
